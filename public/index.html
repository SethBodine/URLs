API Reference

POST /api/shorten

Create a shortened URL

{
  "url": "https://example.com/long/url",
  "customSlug": "my-link"  // optional
}

Response:

{
  "shortUrl": "/xk2a",
  "slug": "xk2a",
  "url": "https://example.com/long/url"
}

Example:

curl -X POST /api/shorten \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com"}'

POST /api/lookup

Look up one or more shortened URLs

// Single lookup
{ "slug": "xk2a" }

// Batch lookup (max 50)
{ "slugs": ["xk2a", "yz9q"] }

Response (single):

{
  "slug": "xk2a",
  "shortUrl": "/xk2a",
  "url": "https://example.com/long/url",
  "createdAt": "2026-04-06T12:00:00.000Z",
  "country": "NZ"
}

Add Authorization: Bearer YOUR_ADMIN_KEY header to also receive ip and userAgent fields.

GET /api/admin

List all shortened URLs (requires authentication)

curl /api/admin \
  -H "Authorization: Bearer YOUR_ADMIN_KEY"

DELETE /api/admin

Delete a link or purge all links (requires authentication)

// Delete one link
curl -X DELETE /api/admin \
  -H "Authorization: Bearer YOUR_ADMIN_KEY" \
  -H "Content-Type: application/json" \
  -d '{"slug":"xk2a"}'

// Purge all links
curl -X DELETE /api/admin \
  -H "Authorization: Bearer YOUR_ADMIN_KEY" \
  -H "Content-Type: application/json" \
  -d '{"purgeAll":true}'
